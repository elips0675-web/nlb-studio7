# Технологии, используемые в проекте "Библиотека 2026"

Этот документ содержит описание основных технологий, которые были использованы при разработке данного проекта.

## Основной стек

*   **[React](https://reactjs.org/)**: JavaScript-библиотека для создания пользовательских интерфейсов. Вся фронтенд-часть приложения построена на React с использованием функциональных компонентов и хуков.

*   **[Vite](https://vitejs.dev/)**: Современный и быстрый сборщик проектов, который обеспечивает мгновенный запуск сервера для разработки и быструю сборку проекта для продакшена.

*   **[TypeScript](https://www.typescriptlang.org/)**: Язык программирования, который расширяет возможности JavaScript, добавляя статическую типизацию. Это помогает избегать ошибок на этапе разработки и улучшает читаемость кода.

*   **[Tailwind CSS](https://tailwindcss.com/)**: Утилитарный CSS-фреймворк, который позволяет быстро создавать современные и адаптивные интерфейсы, используя готовые классы прямо в HTML-разметке.

## Маршрутизация

*   **[React Router](https://reactrouter.com/)**: Библиотека для организации навигации и маршрутизации в React-приложениях. Мы используем ее для создания многостраничного интерфейса.

## Анимации и иконки

*   **[Framer Motion](https://www.framer.com/motion/)**: Библиотека для создания декларативных анимаций в React. Она используется для добавления плавных и красивых переходов и эффектов в интерфейсе.

*   **[Lucide React](https://lucide.dev/)**: Набор простых и красивых иконок в формате SVG, который легко интегрируется в React-проекты.

## Структура проекта

Проект имеет следующую структуру:

-   `src/app/components`: Содержит переиспользуемые React-компоненты (например, `Header`, `Footer`, `News`).
-   `src/app/pages`: Содержит компоненты, которые представляют собой отдельные страницы приложения (например, `Home`, `AllEvents`, `Services`).
-   `src/styles`: Глобальные стили и конфигурация Tailwind CSS.
