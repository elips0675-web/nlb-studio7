# Документация по проекту "nlb-studio7" на 1С-Битрикс

Этот проект представляет собой перенос React-версии сайта на платформу "1С-Битрикс". В качестве основы для верстки используется Bootstrap 4.

## Структура проекта

-   `/` - Корень сайта
    -   `index.php` - Главная страница
    -   `all-news/` - Раздел новостей
    -   `all-events/` - Раздел мероприятий
    -   `services/` - Раздел услуг
    -   `books-catalog/` - Раздел "Электронный каталог"
    -   `interactive-map/` - Раздел "Интерактивная карта"
    -   `.top.menu.php` - Файл верхнего меню
-   `/bitrix_template/` - Папка с шаблоном сайта
    -   `nlb_bootstrap4/` - Шаблон `nlb_bootstrap4`
        -   `header.php` - Шапка сайта
        -   `footer.php` - Подвал сайта
        -   `styles.css` - Стили шаблона

## Кастомизация и наполнение

Управление контентом на сайте предполагается через инфоблоки "1С-Битрикс". Ниже приведены примеры, как можно структурировать данные для наполнения.

### Добавление новостей

Для добавления новостей в инфоблок "Новости" можно использовать следующую структуру полей:

-   **Название:** Заголовок новости (например, *Летний читальный зал открыт!*)
-   **Дата активности:** Дата публикации (например, *15 июня 2024*)
-   **Анонс:** Краткое описание (например, *Наслаждайтесь книгами на свежем воздухе в нашем уютном летнем дворике.*)
-   **Картинка для анонса:** Изображение для новости.
-   **Детальное описание:** Полный текст новости.

**Пример структуры данных из React-проекта:**

```javascript
{
  id: 1,
  title: 'Летний читальный зал открыт!',
  date: '15 июня 2024',
  description: 'Наслаждайтесь книгами на свежем воздухе в нашем уютном летнем дворике.',
  image: 'https://images.unsplash.com/photo-1521587760476-6c12a4b040da?crop=entropy&cs=tinysrgb&fit=max&fm=jpg&ixid=M3w3Nzg4Nzd8MHwxfHNlYXJjaHwxfHxsaWJyYXJ5JTIwb3V0ZG9vcnxlbnwxfHx8fDE3NzUyNjQ1MzF8MA&ixlib=rb-4.1.0&q=80&w=1080',
}
```

### Добавление услуг

Аналогично новостям, услуги добавляются в инфоблок "Услуги".

-   **Название:** Название услуги (например, *Печать на футболках*)
-   **Анонс:** Краткое описание (например, *Футболки и свитшоты с вашим дизайном*)
-   **Детальное описание:** Полное описание услуги.
-   **Дополнительные поля (Свойства инфоблока):**
    -   `ICON`: Иконка услуги
    -   `IMAGES`: Галерея изображений
    -   `BG_COLOR`: Цвет фона для карточки

После добавления элемента в инфоблок, новая услуга должна появиться в общем каталоге, и по клику на нее будет открываться страница с подробным описанием.

**Пример структуры данных для услуги "Печать на футболках":**

```javascript
{
    id: 't-shirts',
    icon: Shirt,
    title: 'Печать на футболках',
    subtitle: 'Футболки и свитшоты с вашим дизайном',
    description: 'Напечатаем любой принт, логотип или фото на качественной одежде из 100% хлопка.',
    fullDescription: 'Предлагаем прямую цифровую печать (DTG) на футболках, свитшотах и худи...',
    images: [
        'https://images.unsplash.com/photo-1576566588028-4147f3842f27?q=80&w=1080',
    ]
}
```

## Развертывание

1.  Разверните "1С-Битрикс" на хостинге.
2.  Настройте подключение к базе данных.
3.  Загрузите файлы текущего проекта в корневую директорию сайта.
4.  В административной панели Битрикса установите шаблон `nlb_bootstrap4` в качестве основного.
5.  Создайте и настройте инфоблоки для новостей, мероприятий, услуг.
6.  Выведите компоненты Битрикса (`news.list`, `catalog` и т.д.) на соответствующих страницах.
