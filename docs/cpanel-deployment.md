# Развертывание админ-панели на cPanel

Развертывание админ-панели на cPanel требует выполнения нескольких шагов. Убедитесь, что на вашем хостинге есть поддержка Node.js.

## 1. Сборка проекта

Перед загрузкой на хостинг, вам нужно собрать админ-панель. Это создаст оптимизированную версию вашего приложения для продакшена.

1.  Перейдите в директорию `admin`:
    ```bash
    cd admin
    ```
2.  Установите все зависимости:
    ```bash
    npm install
    ```
3.  Запустите скрипт сборки:
    ```bash
    npm run build
    ```

После выполнения этой команды у вас появится папка `build` внутри директории `admin`.

## 2. Загрузка файлов на cPanel

1.  Заархивируйте все содержимое папки `admin` в один zip-архив (например, `admin.zip`). **Важно:** в архиве не должно быть папки `node_modules`.
2.  Зайдите в `Файловый менеджер` (File Manager) в вашей cPanel.
3.  Перейдите в корневую директорию вашего сайта (обычно `public_html` или `www`).
4.  Нажмите `Загрузить` (Upload) и выберите ваш `admin.zip`.
5.  После загрузки, кликните правой кнопкой мыши по архиву и выберите `Извлечь` (Extract).

## 3. Настройка Node.js приложения

1.  В главной панели cPanel найдите раздел `Программное обеспечение` (Software) и кликните на `Setup Node.js App`.
2.  Нажмите `Создать приложение` (Create Application).
3.  Заполните форму:
    *   **Node.js version:** Выберите последнюю доступную версию LTS.
    *   **Application mode:** `production`
    *   **Application root:** Укажите путь к вашей папке `admin` (например, `/home/username/public_html/admin`).
    *   **Application URL:** Выберите домен и URL, по которому будет доступна админка (например, `yourdomain.com/admin`).
    *   **Application startup file:** Укажите `dist/server.js`.
4.  Нажмите `Создать` (Create).

## 4. Установка зависимостей на сервере

1.  После создания приложения, на его странице вы увидите опции для управления.
2.  Нажмите на кнопку `NPM Install`. cPanel автоматически определит ваш `package.json` и установит все необходимые зависимости.

## 5. Запуск приложения

1.  После успешной установки зависимостей, нажмите `Запустить приложение` (Start App).

## 6. Настройка базы данных

**Это очень важный шаг!**

Ваше приложение не запустится, если не сможет подключиться к базе данных MongoDB.

1.  Вам нужно создать базу данных MongoDB на вашем хостинге. Обычно это делается через раздел `Базы данных` (Databases) -> `MongoDB`.
2.  После создания базы данных, вы получите строку подключения (connection string). Она выглядит примерно так: `mongodb://user:password@host:port/dbname`.
3.  **Не храните учетные данные в коде!** Лучший способ - использовать переменные окружения.
4.  В настройках вашего Node.js приложения в cPanel есть раздел `Переменные окружения` (Environment Variables).
5.  Добавьте новую переменную:
    *   **Имя:** `MONGODB_URI`
    *   **Значение:** Ваша строка подключения к MongoDB.
6.  Также добавьте переменную для Payload CMS:
    *   **Имя:** `PAYLOAD_SECRET`
    *   **Значение:** Сгенерируйте любую длинную, случайную строку.
7.  Перезапустите ваше приложение, нажав `Restart`.

После выполнения всех этих шагов, ваша админ-панель должна быть доступна по указанному вами URL.
