# Документация по проекту "Библиотека 2026"

## Обзор проекта

Этот проект представляет собой статический веб-сайт для вымышленной "Библиотеки 2026". Сайт разработан с использованием современного стека фронтенд-технологий и демонстрирует различные секции: главную страницу, новости, мероприятия, услуги и каталог печатных услуг.

## Технологии

Веб-сайт создан с использованием следующих технологий:

*   **HTML5:** Для структурирования веб-страниц.
*   **CSS3:** Для стилизации, включая использование кастомных свойств (CSS Variables) для управления темой (цвета, отступы).
*   **Bootstrap 5.3.3:** Как основной CSS-фреймворк для создания адаптивной сетки, компонентов (кнопки, карточки, навигация) и общего стиля. Bootstrap подключается через CDN.
*   **Bootstrap Icons 1.11.3:** Для иконок на сайте. Библиотека также подключается через CDN.
*   **JavaScript (ES6):** Нативный JavaScript используется для интерактивных элементов, таких как слайдер услуг на главной странице.
*   **Unsplash & IBB.co:** Изображения для слайдеров, карточек и фонов были взяты с этих хостингов.

## Структура файлов

Основная логика сайта находится в папке `bootstrap-site/`.

- `index.html`: Главная страница.
- `services.html`: Страница с описанием основных услуг библиотеки.
- `printing.html`: Страница с каталогом печатных услуг.
- `news.html`: Страница с новостями.
- `events.html`: Страница с мероприятиями.

Все страницы используют единый набор стилей, частично определённый в теге `<style>` каждой страницы для простоты демонстрации, и частично заимствованный из Bootstrap.

## Запуск проекта

Поскольку это статический сайт, его можно запустить, просто открыв файл `bootstrap-site/index.html` в любом современном веб-браузере.

Для развертывания проекта использовался внутренний инструмент, аналогичный Firebase Hosting, что подходит для хостинга статических файлов.
