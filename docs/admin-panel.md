# Админ-панель

В качестве админ-панели используется [Payload CMS](https://payloadcms.com/).

## Что сделано

*   **Базовая настройка:** Создана директория `admin` и установлен Payload CMS.
*   **Созданы коллекции:**
    *   `Users`: для управления пользователями.
    *   `Services`: для управления услугами.
*   **Настроены поля:** В коллекциях `Users` и `Services` добавлены базовые поля.
*   **Доступ к админ-панели:** Админ-панель доступна по адресу `http://localhost:3000/admin` после запуска сервера разработки (`npm run dev` в папке `admin`).

## Что осталось сделать

*   **Добавить больше коллекций:** Например, портфолио, отзывы, блог.
*   **Настроить поля в коллекциях:** Добавить, удалить или изменить поля в существующих коллекциях в соответствии с вашими требованиями.
*   **Настроить права доступа:** Более детально настроить права доступа для разных ролей пользователей.
*   **Кастомизировать внешний вид:** Адаптировать дизайн админ-панели под ваш бренд.
